# WordPeek
WordPeek is a lightweight desktop utility that lets you instantly look up word meanings, synonyms, or translations from any app â€” just by selecting text and pressing a global hotkey. It runs quietly in your system tray and works across platforms (Windows, macOS, Linux).

# Python Installation:
Windows: Download from python.org
macOS: Download from python.org or use Homebrew (brew install python)
Linux: Use the system package manager (sudo apt install python3 on Ubuntu, sudo dnf install python3 on Fedora, etc.)
